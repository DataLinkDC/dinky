**在数据开发编辑器内可使用以下快捷键**


| 功能描述       | 快捷键                   | 备注 |
| ---------------- | -------------------------- | ------ |
| 全屏开发       | `F2`                     |      |
| 关闭弹框及全屏 | `Esc`                    |      |
| 保存           | `Ctrl + S`               |      |
| 校验           | `Alt + 2`                |      |
| 美化           | `Alt + 3`                |      |
| 搜索           | `Ctrl + F`               |      |
| 替换           | `Ctrl + H`               |      |
| 撤销           | `Ctrl + Z`               |      |
| 重做           | `Ctrl + Y`               |      |
| 注释           | `Ctrl + /`               |      |
| 缩进           | `选中 + Tab`             |      |
| 取消缩进       | `选中 + Shift + Tab`     |      |
| 选中后续       | `Shift + Alt + Right`    |      |
| 格式化         | `Shift + Alt + F`        |      |
| 编辑多行       | `Ctrl + Shift + Up/Down` |      |
| 复制一行       | `Shift + Alt + Up/Down`  |      |
| 删除一行       | `Ctrl + Shift + K`       |      |
| 匹配下一个     | `Ctrl + F3`              |      |
| 匹配上一个     | `Ctrl + Shift + F3`      |      |
| 前往下一个高亮 | `F7`                     |      |
| 前往上一个高亮 | `Shift +F7`              |      |
| 前面追加换行   | `Ctrl + Shift + End`     |      |
| 追加换行       | `Ctrl + End`             |      |
| 上下换位       | `Alt + Up/Down`          |      |
| 折叠/展开      | `Ctrl + Shift + [/]`     |      |
| 更多快捷键     | `F1`                     |      |
