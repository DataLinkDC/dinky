:::info 信息

在此页面内,可以直观的看到 Flink 任务的基本信息,包含在 Dinky 侧定义的一些配置, 以及远端 Flink 任务的运行状态, Dag 执行图, 任务的 输入输出记录等信息.

并且可以在点击某一Dag 节点后,查看该节点的详细信息. 

以及支持读取该算子的 CheckPoint 信息(注意: 该功能需要 Flink 任务开启 CheckPoint 功能)
:::
