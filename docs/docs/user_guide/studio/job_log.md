## 查看启动和运行日志

在作业为运行状态时，您可以查看其启动日志和运行日志。

|   类型   |                             说明                             |
| :------: | :----------------------------------------------------------: |
| 启动日志 | FlinkSQL 作业启动时的日志，即从FlinkSQL 环境被初始化一直到Job Manager被启动起来中间日志 |
| 运行日志 | Flink的Job Manager和Task Managers的日志可以运维中心查看，也可以在Flink UI界面查看 |

:::tip 说明

启动日志中包含启动作业失败的原因，例如 SQL字段不一致，资源不足等 

:::
